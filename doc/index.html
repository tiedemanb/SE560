Persistance


As stated before, I am using a JDBC connection to an AWS RDS server. In my pictures, I have shown inputs from WFetch, which could make it easy for you to replicate (or add more) data in the current system. If you would like, I also have an admin page at http://se560btiedema.elasticbeanstalk.com/admin/reset that will wipe the database clean, to help test persistance. Use this last though, unless you want to repopulate everthing.
